# Function of net loss over time from malariasimulation Net loss is exponential.

Function of net loss over time from malariasimulation Net loss is
exponential.

## Usage

``` r
net_loss_exp(t, mean_retention)
```

## Arguments

- t:

  Single value or vector of timesteps in days.

- mean_retention:

  (Country-specific) average duration of net retention in days.

## Value

The proportion of nets retained over time.
