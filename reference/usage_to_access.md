# Convert usage to access

Convert usage to access

## Usage

``` r
usage_to_access(usage, use_rate, max_access_value = 1)
```

## Arguments

- usage:

  A single value or vector of desired target usages to model.

- use_rate:

  A single value or vector of usage rates.

- max_access_value:

  Value to use if resulting access \> 1

## Value

Access
